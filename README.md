# Calculator_JS

A modern calculator built with HTML, CSS, and JavaScript that performs basic arithmetic operations through a simple and user-friendly interface.

## About the Project

This project was created to practice JavaScript fundamentals, DOM manipulation, event handling, and responsive web design. The calculator allows users to perform everyday mathematical calculations quickly and efficiently.

## Features

- Addition, subtraction, multiplication, and division
- Interactive button-based interface
- Clear and delete functionality
- Real-time display updates
- Responsive design for different screen sizes
- Clean and modern UI

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap
